# Security Policy

## Reporting a Vulnerability

Please report security issues via GitHub Security Advisories for this
repository. If advisories are not enabled, open an issue with minimal details
and request a private channel for disclosure.

Include the following when possible:

- Affected version or commit
- Steps to reproduce
- Impact assessment
- Logs or backtraces (redact sensitive data)

## Scope

In scope:

- `ota-fetch` source code and build system
- Test scripts and fixtures in `test/`

Out of scope:

- Vulnerabilities in third-party dependencies (report upstream)
